# Interfaz de tutoriales y comentarios

Interfaz web mínima desarrollada con Angular para consumir la API REST del
proyecto.

## Funcionalidades

- Registrar tutoriales.
- Listar y seleccionar tutoriales.
- Registrar comentarios para el tutorial seleccionado.
- Listar los comentarios de un tutorial.
- Reemplazar el contenido de un comentario mediante `PUT`.
- Eliminar comentarios.

## Requisitos

- Node.js 24 o compatible.
- API REST ejecutándose en `http://127.0.0.1:8000`.

## Instalación

```powershell
npm install
```

## Ejecución

```powershell
npm start
```

La interfaz queda disponible en:

```text
http://localhost:4200
```

## Compilación

```powershell
npm run build
```
