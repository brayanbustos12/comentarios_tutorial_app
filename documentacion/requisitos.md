# Requerimientos

## Tutoriales

El sistema debe permitir:

- Registrar el título de un tutorial.
- Registrar la descripción del tutorial.
- Registrar la fecha de publicación.
- Listar los tutoriales registrados.

## Comentarios

El sistema debe permitir:

- Registrar comentarios asociados a un tutorial.
- Listar los comentarios de un tutorial.
- Modificar el contenido de un comentario.
- Eliminar un comentario.

## Requerimientos técnicos

- Implementar una API REST.
- Utilizar un framework para el servidor.
- Garantizar la persistencia de la información.
- Separar presentación, negocio y persistencia.
- Documentar la instalación y ejecución.
- Incluir un diagrama del modelo de datos.
- Incluir pruebas automatizadas.
- Incluir evidencia de funcionamiento.
- Mantener una traza del desarrollo con Git y GitHub.
