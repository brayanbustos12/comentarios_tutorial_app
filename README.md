# Aplicación de tutoriales y comentarios

Prueba técnica para desarrollar una aplicación web que permita gestionar
tutoriales y sus comentarios.

## Estado del proyecto

En construcción.

## Funcionalidades

- Registrar tutoriales.
- Listar tutoriales.
- Registrar comentarios para un tutorial.
- Listar los comentarios de un tutorial.
- Modificar comentarios.
- Eliminar comentarios.

## Tecnologías

- Python 3.13.
- FastAPI.
- PostgreSQL 18.
- SQLAlchemy.
- Alembic.
- Angular.

## Estructura

- `servidor`: API REST y lógica del negocio.
- `interfaz`: aplicación web Angular.
